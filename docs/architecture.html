ESP32-S3 LSTM Inference Architecture

32.59 tok/s char-LSTM on $4 hardware — ESP-NN SIMD + SRAM tiling + dual-core

Optimizations

  • ESP-NN SIMD dot for LSTM gates (novel — ESP-NN was CNN-only)
  • SRAM weight tiling: PSRAM -> internal SRAM (7.6x faster whh)
  • Dual-core gate parallelism via FreeRTOS semaphores
  • Fixed-point quantization with sigmoid/tanh LUT
  • 16-byte alignment for SIMD precondition

Hardware

  • ESP32-S3 WROOM N8R8 ($4)
  • Dual-core Xtensa LX7 @ 240MHz
  • 8MB octal PSRAM (~100 MB/s)
  • 512KB SRAM (~200+ MB/s)
  • ESP-NN vector instructions (16 int8 lanes)
